# REGISTRO DE ESTUDIANTES

- Aplicacion web para registro de estudiantes de la universidad
- La aplicacion permite registrar a varios estudiantes al mismo tiempo y guardar su entrada y salida en un solo lugar.
- La aplicacion permite generar reportes automaticos diarios o generar reportes manualmente.

## ¿Por que utilizar esta herramienta? 

- Para ya no usar hojas que despues se tiran

## Desarrollo

- Esta aplicacion se creo utilizando distintas herramientas como: NodeJs, Supabase, HTML & CSS
- HTML & CSS: 
    - Estos lenguajes se utilizan para desarrollar el frontend para que los estudiantes puedan ingresar sus datos y enviarlos al backend.
- NodeJs:
    - NodeJs se utiliza para el backend, NodeJs recibe los datos enviados desde el fronend y con middlewares/filtros almacenar los datos en supabase.
- Supabase:
    - Supabase es una base de datos que se utiliza para almacenar los datos del backend, Supabase guarda los datos de manera ordenada segun el tipo de dato ingresado.
